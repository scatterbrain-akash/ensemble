The one-line pitch: a bounded multi-agent pipeline that turns a synthetic denial letter into an evidence-grounded appeal package — or a reason-coded human escalation when facts or evidence are missing. Zero PHI, purpose-based model routing (frontier at work / free-tier at home), and inspectable traces throughout.
v1 → v2 headline: merged the two source docs, corrected the CMS API assumption (LCD endpoints need a free license token), closed the prompt/eval/reproducibility gaps, and added a bounded state machine, capability-tier routing, an ADR register, a threat model, and cost governance. See the full breakdown in Solution Design §1.
