# Changelog

## [1.0.0] - 2026-07-10
### Added
- CMSCoverageTool mapped to CMS Coverage API endpoints with license token flow and retry/backoff.
- Fixture retrieval fallback and deterministic mock LLM provider used for tests and deterministic runs.
- Tiered caching with optional Redis and file-backed fallback.
- Cost tracking, tracing primitives, and a basic CI workflow.
- Unit and integration tests covering core retrieval and token flows.
